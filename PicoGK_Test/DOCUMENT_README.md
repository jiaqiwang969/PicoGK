## 核心内容亮点

### 数学基础

1. **体素离散化公式**：
   ```
   (x, y, z) → (⌊x/Δ⌋, ⌊y/Δ⌋, ⌊z/Δ⌋)
   ```

2. **有符号距离场定义**：
   ```
   φ(x) = { -d(x, ∂Ω)  if x ∈ Ω
          {  0          if x ∈ ∂Ω
          { +d(x, ∂Ω)  if x ∉ Ω
   ```

3. **Gyroid 三周期极小曲面**：
   ```
   sin(x)cos(y) + sin(y)cos(z) + sin(z)cos(x) = 0
   ```

### 算法复杂度

- **布尔运算**：O(N₁ + N₂)
- **偏移操作**：O(N)
- **网格化**：O(N)
- **点查询**：O(log N) 或 O(1)
